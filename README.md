# جزوه درس پایگاه داده (مقطع کارشناسی ارشد)

## معرفی
این مخزن شامل جزوه‌ها و مطالب آموزشی درس **پایگاه داده** در مقطع کارشناسی ارشد می‌باشد.  
مطالب این مجموعه با هدف جمع‌آوری، سازمان‌دهی و مرور مفاهیم پیشرفته پایگاه داده تهیه شده است.

## فهرست مطالب

### فصل ۸: انواع داده‌های پیچیده (Complex Data Types)
- بررسی انواع داده‌های پیچیده
- ساختارهای داده‌ای پیشرفته در پایگاه داده
- مفاهیم مرتبط با داده‌های غیرساده

### فصل ۹: توسعه برنامه‌های کاربردی (Application Development)
- توسعه برنامه‌های کاربردی مبتنی بر پایگاه داده
- ارتباط برنامه‌ها با سیستم مدیریت پایگاه داده
- مفاهیم مرتبط با طراحی و پیاده‌سازی برنامه‌های کاربردی

### فصل ۱۰: داده‌های بزرگ (Big Data)
- مفهوم داده‌های بزرگ
- ویژگی‌های Big Data
- چالش‌های ذخیره‌سازی و پردازش داده‌های حجیم
- فناوری‌ها و رویکردهای مرتبط با داده‌های بزرگ

### فصل ۱۱: تحلیل داده (Data Analytics)
- مفاهیم تحلیل داده
- روش‌های استخراج و تحلیل اطلاعات از داده‌ها
- کاربرد تحلیل داده در سیستم‌های پایگاه داده

### فصل ۱۴: نمایه‌سازی (Indexing)
- مفهوم Index در پایگاه داده
- روش‌های ایجاد و مدیریت نمایه‌ها
- تأثیر نمایه‌سازی بر بهبود کارایی پرس‌وجوها

- **مقطع:** کارشناسی ارشد
- **تهیه‌کننده:** تهیه و تنظیم توسط من
