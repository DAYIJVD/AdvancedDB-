# جزوه درس پایگاه داده (مقطع کارشناسی ارشد)

## معرفی
این مخزن شامل جزوه‌ها و مطالب آموزشی درس **پایگاه داده** در مقطع کارشناسی ارشد می‌باشد.  
مطالب این مجموعه با هدف جمع‌آوری، سازمان‌دهی و مرور مفاهیم پیشرفته پایگاه داده تهیه شده است.

## فهرست مطالب

### فصل ۸: انواع داده‌های پیچیده (Complex Data Types)
- داده‌های نیمه‌ساختاریافته
- شیءگرایی (Object Orientation)
- داده‌های متنی
- داده‌های مکانی

### فصل ۹: توسعه برنامه‌های کاربردی (Application Development)
- برنامه‌های کاربردی و رابط‌های کاربری
- تکامل معماری برنامه‌های کاربردی
- معماری وب

### فصل ۱۰: داده‌های بزرگ (Big Data)

-  پرس‌وجو از داده‌های بزرگ 
- سیستم‌های ذخیره‌سازی داده‌های بزرگ
- پارادایم MapReduce
- فراتر از MapReduce — Spark
- داده‌های جریانی
- پایگاه‌های داده گراف

### فصل ۱۱: تحلیل داده (Data Analytics)
- انبار داده    
- پردازش تحلیلی آنلاین
- داده‌کاوی  

### فصل ۱۴: نمایه‌سازی (Indexing)
- مفاهیم پایه
- نمایه‌های مرتب (Ordered Indices)
- فایل‌های نمایه درخت B+
